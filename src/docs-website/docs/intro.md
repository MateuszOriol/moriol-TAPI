# TAPI Introduction

API providing information on The Witcher univers - characters locations and monsters

## Technologies Used

- **Backend**:
  - Node.js with Express
  - GraphQL for flexible queries
  - REST API for traditional endpoints

- **Documentation**:
  - OpenAPI/Swagger
  - Docusaurus
